SLIDE 6: KEY FINDINGS + ROBUSTNESS DEMO
================================================================================
Visual Layout: 3-Part Slide (Findings + 2 Charts)

LEFT SIDE - 5 KEY FINDINGS (Text boxes):
1. ✓ Combination Methods Win
   Dropout+BatchNorm +0.91% over Baseline
   → Complementary mechanisms (masking + normalization)

2. ✓ Adaptive > Momentum-Based
   AdamW: 36 epochs → SGD: 65 epochs
   → 1.8x speedup, same accuracy

3. ✓ Gap = Overfitting Proxy
   Smallest gap (0.18) = best generalization
   → Diagnostic metric for method selection

4. ✓ Sparsity ≠ Accuracy
   L1: 38.7% sparsity, only 87.91% accuracy
   → Don't optimize for wrong objective

5. ✓ Standard Regularization ≠ Adversarial Robustness
   Need specialized training for true robustness

TOP RIGHT VISUAL:
📊 IMAGE: "Adversarial Robustness Comparison (FGSM ε=0.05)"
File: results/fgsm_robustness_comparison.png
Shows: Clean Data (dark) vs FGSM Attacked (striped) accuracy across all 10 variants
KEY NUMBERS:
  • Dropout+BatchNorm: 81.43% under attack (vs 89.15% clean) = -7.68% drop
  • Baseline: ~75% under attack = -13% drop
  • L1: ~80% under attack
  → DropoutBatchNorm slightly MORE robust (+3-5% resistance)

BOTTOM RIGHT VISUAL:
📊 IMAGE: "Weight Distribution Histograms - Regularization Effect"
File: results/weight_histograms.png
Shows: Multi-layer (Layer 1→3) and output layer weight distributions for all 10 variants
VISUAL INSIGHTS:
  • Baseline (RED): Broad, uniform distribution → Dense weights
  • L1 (BLUE): Sharp spike at 0 → Induced zeros
  • L2 (GREEN): Narrower than baseline → Regularized magnitudes
  • Dropout/BatchNorm: Similar to baseline → Indirect effects
  • Combined methods: Blended patterns visible

KEY INSIGHT:
Different regularizers act on different weight aspects:
  - L1/L2: Weight magnitude control (visible in histograms)
  - Dropout: Stochastic masking (not visible in final weights)
  - BatchNorm: Activation normalization (not visible in weight histograms)
